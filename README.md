D is a simple standalone httpd.

## Usage

1. Install it `cargo install d` (or `cargo install --git https://github.com/liuchong/d` for the newest)

2. Cd to somewhere and run it

``` bash
cd ~/somewhere

# check usage
d -h

# run it
d
# or
d -p 8888
```
